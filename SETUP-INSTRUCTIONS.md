# GitHub Pages Setup - Step by Step

## Quick Setup (5 minutes)

### 1. Create Repository
1. Go to github.com
2. Click "+" → "New repository"
3. Name: `ai-guides` (or whatever you want)
4. Make it **Public**
5. Check "Add a README file"
6. Click "Create repository"

### 2. Enable GitHub Pages
1. In the repo, click "Settings"
2. Scroll to "Pages" (left sidebar)
3. Under "Source", select **main** branch
4. Click "Save"
5. Wait 1-2 minutes

Your site will be live at:
`https://[your-username].github.io/ai-guides/`

### 3. Upload Files
I'll prepare all files in the correct structure. Then you either:
- **Option A:** Drag and drop into GitHub web interface
- **Option B:** Clone repo and push files
- **Option C:** Give me collaborator access and I'll push directly

## Custom Domain (Optional)
If you have codsworthphd.com:
1. In repo Settings → Pages
2. Add custom domain: `guides.codsworthphd.com`
3. In your DNS settings, add CNAME record:
   - Name: `guides`
   - Value: `[username].github.io`
