# Color Palette - Subtle Industry Accents

## Base Colors (Keep)
- Primary: #000000 (black)
- Background: #FFFFFF (white)
- Secondary text: #666666
- Border: #E5E7EB

## Industry Accent Colors (Subtle, Professional)

### Roofing
- Accent: #C1440E (burnt orange - roof tile color)
- Used for: badges, hover states

### HVAC
- Accent: #1E40AF (deep blue - cool/professional)
- Used for: badges, hover states

### Dental
- Accent: #059669 (medical green - trust/health)
- Used for: badges, hover states

### Law
- Accent: #7C2D12 (burgundy - traditional law)
- Used for: badges, hover states

### Real Estate
- Accent: #0891B2 (teal - homes/property)
- Used for: badges, hover states

## Implementation
Each product card will have its own subtle accent color for:
- Badge background
- Hover border highlight
- Button hover state

Keeps professional while adding visual interest without looking generic.
